# Play Console Closed Test Upload Steps

Use placeholder-only docs in the repository. Real tester emails, reviewer
credentials, and private support details belong only in Play Console or private
local tracking.

## Steps

1. Create the app in Play Console.
2. Set app name: `Match A Man`.
3. Select the default language.
4. Fill app details, category, and contact details.
5. Add the privacy policy URL.
6. Add the account deletion URL if required.
7. Fill Data Safety.
8. Fill Content Rating.
9. Fill Target Audience.
10. Add app access instructions if login is required.
11. Create the closed testing track.
12. Add tester emails or a Google Group.
13. Upload the signed `app-release.aab`.
14. Enter release notes.
15. Review Play Console warnings/errors.
16. Submit the closed test release.
17. Send the opt-in link to testers.
18. Track testers and feedback privately.

## Notes

- Do not upload a debug APK.
- Do not commit tester emails.
- Do not commit reviewer passwords.
- Do not overclaim safety, verification, or event supervision.
- Match A Man should be positioned as a sports/social events app, not a dating
  app.
