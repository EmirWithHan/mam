# Feature Roadmap

## Visual And Social Event Layer

- Event-linked feed posts now have a sport type tag when the linked event exposes a sport type.
- Event cards and event detail headers use automatic cover visuals derived from sport type; users do not upload event images in this slice.
- Profile badge UI foundation exists with earned, locked, and upcoming states plus a "Tümü gör" view.
- The full trust-score and badge earning algorithm is intentionally deferred to a later prompt.
- Business accounts, sponsored event logic, check-in, phone verification, SMS/OTP, and push notifications remain future prompts.
