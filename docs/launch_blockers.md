# Launch Blockers

## Public Launch Blockers

- Facebook/Meta app is not submission-ready until icon, Privacy Policy URL, Data Deletion URL/instructions, category, and app mode are completed.
- Facebook public use requires Live Mode; Development Mode is limited to roles/test users.
- Apple login is postponed.
- Push notifications are postponed.
- Production app icon/splash should be verified across Android, iOS, and web.
- Privacy Policy, Terms of Service, and Data Deletion pages are needed.
- Store screenshots and listing copy are needed.
- Android release signing is not configured.
- iOS Apple Developer/signing/provisioning is not configured.
- Admin/moderation dashboard is missing.
- Production analytics/crash reporting is postponed.
- Production rate limiting and abuse detection need follow-up before larger public launch.

## Closed Beta Notes

- Closed beta can proceed only with clear tester expectations.
- Meta/Facebook testing can use roles/test users before Live Mode.
- Apple login should remain disabled.
- Push/realtime notifications should not be promised.
