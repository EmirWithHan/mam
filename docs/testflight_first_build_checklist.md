# TestFlight First Build Checklist

Use this after the IPA upload succeeds.

## After Upload

- [ ] Wait for build processing.
- [ ] Select build in App Store Connect.
- [ ] Add internal testers.
- [ ] Install through TestFlight.
- [ ] Record iPhone model.
- [ ] Record iOS version.
- [ ] Screenshot any bugs.

## Internal Smoke Test

- [ ] Launch.
- [ ] Register.
- [ ] Email confirmation link.
- [ ] Username onboarding.
- [ ] Home.
- [ ] Events.
- [ ] Event detail.
- [ ] Create.
- [ ] Social.
- [ ] Profile.
- [ ] Search.
- [ ] Settings.
- [ ] Forgot password.
- [ ] Account deletion/request path if implemented.
- [ ] No raw Supabase/Postgrest errors in UI.
- [ ] No visible overflow on tested iPhone.

Do not invite external testers until the internal smoke test passes.
