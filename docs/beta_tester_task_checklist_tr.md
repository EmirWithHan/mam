# Beta Tester Task Checklist TR

1. Uygulamayı kur.
2. Gerçek e-posta ile kayıt ol.
3. E-postanı doğrula.
4. Kullanıcı adı seç.
5. Ana Akış ekranını kontrol et.
6. Etkinlikler ekranını aç.
7. Bir etkinlik detayına gir.
8. Etkinlik oluşturmayı dene.
9. Profil ekranını aç.
10. Kullanıcı aramayı dene.
11. Ayarlar ekranını aç.
12. Rahatsan Şifremi unuttum akışını dene.
13. Hata görürsen ekran kaydı veya ekran görüntüsü gönder.

Lütfen şifreni, doğrulama linkini veya şifre sıfırlama linkini gönderme.
