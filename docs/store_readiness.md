## Long Description Draft

Match A Man is an event-centered social sports app. Discover local football,
basketball, tennis, running, and social activity events, request to join, chat
after approval, and share moments from events on your profile.

The app is designed for sports, activities, trust, and real-life social plans.
It is not a dating app.

Current in-app user-facing copy is locked to Turkish for MVP screens. Store
listing copy may still be localized separately before submission.

MVP safety tools include profile privacy, follow requests, report actions,
blocking, admin business review, feedback, and trust-score signals for event
behavior.

## Keywords Draft

running, meetup, group events, activity friends

## Android Release Build

Play Store upload uses the signed release AAB, not a debug APK. Build it with
dart-defines and placeholder-only docs/scripts:

If the Android `applicationId` changes, rebuild the signed AAB before upload.
Do not upload an artifact built with an older id.

```powershell
flutter build appbundle --release `
  --dart-define=SUPABASE_URL="YOUR_SUPABASE_URL" `
  --dart-define=SUPABASE_ANON_KEY="YOUR_SUPABASE_ANON_KEY"
```

Expected output:

```text
build/app/outputs/bundle/release/app-release.aab
```

Signed AAB prep is documented in `docs/android_signed_aab_final_prep.md`.
Local release signing requires:

```text
android/key.properties
android/app/upload-keystore.jks
```

Generate the upload keystore manually:

```powershell
keytool -genkey -v -keystore android/app/upload-keystore.jks -keyalg RSA -keysize 2048 -validity 10000 -alias upload
```

Create `android/key.properties` manually:

```properties
storePassword=YOUR_STORE_PASSWORD
keyPassword=YOUR_KEY_PASSWORD
keyAlias=upload
storeFile=../app/upload-keystore.jks
```

These files are local secrets and must never be committed. Play App Signing
enrollment, Play Console upload, Data Safety, content rating, store listing,
privacy policy URL, and account deletion URL remain manual steps.

Current version is `1.0.0+1` (`versionName` `1.0.0`, `versionCode` `1`).
Every Play upload must increase `versionCode`.
